
# Comparative Hands-On Evaluation of 15 Endpoint Security Products Across 6 Real-World Attack Scenarios
## Overview

This research presents a practical hands-on evaluation of 15 endpoint security products against six attack scenarios designed to represent key stages of the modern attack lifecycle.

The objective was to evaluate prevention, detection, post-incident visibility, and defensive effectiveness under controlled laboratory conditions.

## Evaluated Scenarios

* USB HID Attack
* Traditional HTTP Command and Control
* Cloud-Based Command and Control
* Traditional Ransomware
* IFRET (Indirect File Replacement Encryption Technique)
* Remote Encryption Workflow

## Key Findings

* Strong protection against traditional ransomware behavior.
* Reduced visibility across cloud-based command-and-control workflows.
* Significant challenges identifying IFRET-style activity.
* Limited visibility when encryption operations occur outside the monitored endpoint.
* Defense-in-depth remains essential for modern enterprise security architectures.

## Disclaimer

This research was conducted independently in a controlled laboratory environment. Results reflect observations made during testing and should not be interpreted as absolute measurements of product effectiveness in all environments.
